# Project Instructions

## Project Overview

- This is a WooCommerce-based WordPress plugin that enables date and time selection on products to make products bookable.
- Keep all implementation aligned with WooCommerce and WordPress architecture and extension standards.

## Development Standards

- Always follow WooCommerce and WordPress best practices for code quality, maintainability, performance, and security.
- Use WordPress and WooCommerce hooks, filters, APIs, and helper functions for maximum compatibility with themes and third-party plugins.
- Avoid overriding templates or implementing hard-coded theme-dependent behavior unless explicitly required.
- Keep functionality modular and organized using object-oriented programming (OOP) principles.
- Validate, sanitize, and escape all input/output following WordPress security standards.
- Use nonces and capability checks where applicable.
- Prefer backward-compatible, minimally invasive changes.

## Compatibility Principles

- Ensure compatibility with WooCommerce simple and variable products.
- Preserve compatibility with popular themes, page builders, and WooCommerce checkout/cart flows.
- Ensure features are compatible with current WooCommerce conventions, including HPOS-safe patterns where applicable.
